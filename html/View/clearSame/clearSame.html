使用說明

1. 將資料貼到 A / B / C 欄位，每行視為一個項目。

2. A 有、B 沒有：顯示只出現在 A、但沒出現在 B 的項目。

3. B 有、A 沒有：顯示只出現在 B、但沒出現在 A 的項目。

4. C 去重：會自動移除 C 欄位中重複的項目，只保留第一次出現。

5. 勾選 不區分大小寫，比對時會把英文字轉成同一種大小寫再比較。

6. 下方結果區塊都可以一鍵 複製，方便貼回 Excel 或其他系統。

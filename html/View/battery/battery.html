電池空值回補

asset_no 外觀車號
lms_maintain_lists 回補
battery_id 電池

電池空值處理流程

1. 查詢電池單
維調提供 asset_no，於 lms_maintain_lists 查詢該車電池維修單： {"asset_no":"xxxxxxx","maintain_id":{"$in":[3,4,9]},"bike_type":"02","type_id":4}

2. 查看 Core battery_id
‧ 有 → 可回補
‧ 空值 → 請維調重啟車機
（不建議從 webapi.log 取值）

3. 回補至 LMS
{ "asset_no":"xxxxxxx", "process":NumberInt(0), "type_id":NumberInt(4) }
